# n++ game

**goal of the game**

survive for a time duration and find key and goto door.

**mechanics**

- 🚀rockets - enemy
- 🐹player - protagonist
    - meele attacks (to destroy rockets)
- 🔥powerups - some special abilities/weapons
- 🏢levels - with stairs
- ⏳timer - try to survive till some time
- 🔑key - pick up after you survived
- 🚪door - open door after picking up key
